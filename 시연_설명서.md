# 📱 폰화면 보기 - 강의 시연 설명서

---

## 1. 이 프로그램은 뭔가요?

강의할 때 **PC/노트북 화면에 폰 모양으로 사이트를 보여주는 도구**입니다.

수강생이 전원 모바일이라면, 강사가 PC에서 Gemini나 ChatGPT 같은 사이트를 열어도 수강생 폰에서 보이는 화면과 다르게 나옵니다. 이 프로그램을 쓰면 **PC 화면에 폰 화면 그대로** 보여줄 수 있어서, 수강생이 보고 바로 따라할 수 있습니다.

### 네 가지 메뉴

| 메뉴 | 설명 |
|---|---|
| **📱 폰화면** | PC에서 사이트를 열면 → 폰 크기 새 창으로 모바일 화면처럼 보여줌 |
| **📲 앱 설치** | 수강생이 폰에 앱 설치할 때 → 스토어 바로가기 + 설치 순서 안내 |
| **🔗 폰 띄우기** | 강사 폰 화면을 프로젝터에 직접 띄우는 방법 안내 |
| **📖 사용법** | 노트북 세팅, 핫스팟 연결 등 강의 준비 가이드 |

---

## 2. 시작하기 전 준비

### 필요한 것
- **노트북 또는 PC** (인터넷 연결 필요)
- **프로젝터** 또는 대형 모니터 (노트북에 연결)
- **웹 브라우저** (Chrome, Edge 등)

### 프로그램 열기
**방법 1 - 파일로 바로 열기:**
- 이 폴더의 `index.html` 파일을 **더블클릭**하면 브라우저에서 열립니다

**방법 2 - 인터넷에 올려서 쓰기 (추천):**
1. **Netlify**(무료)에 이 폴더를 올리면 주소가 생김
2. 어떤 PC에서든 그 주소로 접속하면 바로 사용 가능
3. 아래 "배포 방법" 참고

---

## 3. 강의실 노트북 세팅 순서

```
1. 노트북을 프로젝터/대형 모니터에 연결
2. 브라우저(Chrome 추천)에서 이 프로그램 열기
3. [📲 앱 설치] 탭에서 수강생 앱 설치 안내 (처음 한 번만)
4. [📱 폰화면] 탭으로 전환
5. "자주 쓰는 도구"에서 도구 클릭 → 폰 크기 새 창 열림
6. 열린 새 창을 프로젝터로 보여주며 실습 진행!
```

## 4. 📱 폰화면 보기 (메인 기능)

### 사이트 열기
**방법 A - 자주 쓰는 도구 누르기 (빠름):**
1. 왼쪽 **"자주 쓰는 도구"** 에서 원하는 도구를 클릭
2. Gemini, ChatGPT, Claude, Suno, Canva 등이 미리 등록되어 있음
3. 클릭하면 **폰 크기의 작은 브라우저 창**이 바로 열림

**방법 B - 주소 직접 입력:**
1. **"사이트 주소"** 입력란에 주소 입력 (예: `gemini.google.com`)
2. **"열기"** 버튼 클릭

### 열기 방식 선택
| 방식 | 설명 |
|---|---|
| **📱 폰 크기 새 창 (추천)** | 폰 크기의 브라우저 창이 새로 열림. **모든 사이트 동작** |
| 🖼️ 프레임 안에 열기 | 폰 프레임 안에 넣지만, 대부분 사이트가 차단됨 |

> 💡 **팁:** 처음 열 때 브라우저가 팝업을 차단할 수 있어요. 주소창 옆 팝업 차단 아이콘 → "허용"을 눌러주세요.

### 수강생에게 보여주기
1. 열린 폰 크기 새 창을 프로젝터 화면에 배치
2. "여기를 눌러보세요" 등으로 안내
3. 수강생은 자기 폰에서 같은 사이트에 접속해서 따라함

## 5. 📲 앱 설치 안내 (수강생용)

강의 시작할 때 수강생이 폰에 앱을 설치해야 할 경우 사용합니다.

### 사용법
1. 왼쪽 패널에서 **"📲 앱 설치"** 탭을 선택
2. 화면에 각 도구별 설치 카드가 나옵니다:
   - **🍎 아이폰 설치** 버튼 → App Store로 바로 이동
   - **🤖 안드로이드 설치** 버튼 → Play Store로 바로 이동
   - **🌐 웹으로 접속** 버튼 → 앱 설치 없이 브라우저로 사용
3. 각 카드에 **설치 순서**가 단계별로 적혀 있음
4. 프로젝터에 이 화면을 띄우고 수강생이 따라하도록 안내

### 지원 도구 목록
| 도구 | 앱 설치 | 웹 접속 |
|---|---|---|
| Gemini | 아이폰 / 안드로이드 | gemini.google.com |
| ChatGPT | 아이폰 / 안드로이드 | chatgpt.com |
| Suno | 아이폰 / 안드로이드 | suno.com |
| Canva | 아이폰 / 안드로이드 | canva.com |
| Perplexity | 아이폰 / 안드로이드 | perplexity.ai |
| Gamma | 앱 없음 | gamma.app |

> 💡 **팁:** 대부분의 AI 도구는 앱 설치 없이 **웹 브라우저에서도** 사용 가능합니다. 수강생에게 "앱 설치가 어려우면 브라우저로 접속하세요"라고 안내하면 됩니다.

## 6. 📶 와이파이가 안 될 때 - 핫스팟 연결

강의실 보안 와이파이가 안 되거나 인터넷이 없을 때, **강사 폰의 핫스팟**으로 해결할 수 있어요.

### 아이폰 핫스팟 켜기
1. **설정** → **개인용 핫스팟** 열기
2. **"다른 사람의 연결 허용"** 켜기
3. 와이파이 이름과 암호가 화면에 표시됨
4. 수강생에게 이름/암호를 알려주고 연결하도록 안내

### 안드로이드 핫스팟 켜기
1. 화면 위에서 아래로 내려서 **빠른 설정** 열기
2. **"모바일 핫스팟"** 또는 **"테더링"** 버튼 누르기
3. 핫스팟 이름과 비밀번호를 수강생에게 안내

### 노트북도 핫스팟에 연결하기
1. 강사 폰에서 핫스팟 켜기
2. 노트북 와이파이에서 **핫스팟 이름 찾아서 연결**
3. 노트북 브라우저에서 이 프로그램 열기
4. 수강생도 같은 핫스팟에 연결
5. 모두 인터넷 사용 가능! → 실습 진행

### 핫스팟 사용 팁
- 데이터 요금제가 충분한지 먼저 확인하세요
- 수강생이 10명 이상이면 데이터 소모가 클 수 있어요
- 영상 재생은 피하고, 텍스트 위주 실습을 추천해요
- 무제한 요금제라면 걱정 없이 사용 가능!
- 보조 배터리를 준비하면 더 안심이에요

## 7. 🔗 내 폰 화면 직접 띄우기 (미러링)

폰에서 직접 실습하면서 화면을 보여줘야 할 때 사용합니다.

### 사용법
1. 왼쪽 패널에서 **"🔗 폰 띄우기"** 탭 선택
2. 아이폰/안드로이드별 연결 방법 안내가 나옵니다

### 아이폰
- 같은 와이파이 연결 → 제어센터 → 화면 미러링 → PC 선택
- Windows PC에는 **AirServer** 또는 **LonelyScreen** 설치 필요

### 안드로이드
- **무선:** 빠른 설정 → Smart View / 화면 미러링 → PC 선택
- **유선 (추천):** USB 케이블 + **scrcpy**(무료 프로그램) 사용

## 8. 화면 돌리기 / 폰 종류 / 도구 관리

### 화면 돌리기
- 왼쪽 패널 **"화면 돌리기"**에서 **세로** 또는 **가로** 선택
- 영상 시청 같은 경우 가로로 돌려서 보여줄 수 있음

### 폰 종류 변경
- **iPhone 14** - 아이폰 사용자가 많을 때
- **Galaxy S24** - 갤럭시 사용자가 많을 때
- **iPhone SE** - 작은 화면
- **iPad Mini** - 태블릿 화면

### 자주 쓰는 도구 추가/삭제
**추가:**
1. 왼쪽 **"도구 목록 편집"** 에서 이름과 주소 입력
2. **"넣기"** 버튼 클릭

**삭제:**
- 도구 목록에서 **×** 버튼 클릭

## 9. 강의 시연 시나리오 예시

### 예시 1: 앱 설치 + Gemini 실습
```
1. [📲 앱 설치] 탭에서 Gemini 설치 안내 화면을 프로젝터에 띄움
2. "아이폰이신 분은 '아이폰 설치' 버튼, 갤럭시는 '안드로이드 설치' 눌러주세요"
3. 수강생 앱 설치 완료 확인
4. [📱 폰화면] 탭으로 전환 → "자주 쓰는 도구"에서 Gemini 클릭
5. 폰 크기 새 창에서 Gemini 사용법을 단계별로 시연
6. "여기 입력란에 질문을 입력해보세요" → 수강생이 따라함
```

### 예시 2: 핫스팟 연결 + Suno 실습
```
1. 강의실 와이파이 안 됨 → 강사 폰 핫스팟 켜기
2. [📖 사용법] 탭에서 핫스팟 연결 방법을 프로젝터에 띄움
3. 수강생에게 핫스팟 이름/비밀번호 안내
4. 노트북도 핫스팟에 연결
5. [📲 앱 설치] 탭에서 Suno 설치 안내
6. [📱 폰화면] 탭에서 Suno 열기 → 음악 만들기 시연
```

### 예시 3: 여러 도구 비교 강의
```
1. "자주 쓰는 도구"에서 Gemini 클릭 → 질문 입력 → 결과 보여주기
2. 주소란에 chatgpt.com 입력 → 같은 질문 → 결과 비교
3. Claude도 동일하게 → 세 도구의 답변 차이를 설명
```

### 예시 4: 폰 미러링으로 앱 설치 과정 직접 보여주기
```
1. [🔗 폰 띄우기] 탭에서 미러링 방법 확인
2. 강사 폰을 노트북에 미러링 연결
3. 프로젝터에 강사 폰 화면이 그대로 나옴
4. 앱스토어 열기 → Suno 검색 → 설치 과정을 직접 보여줌
5. 수강생이 프로젝터 보면서 자기 폰에서 따라서 설치
```

## 10. 배포 방법 (인터넷에 올리기)

### Netlify 사용 (가장 쉬움, 무료)
1. https://netlify.com 에 가입합니다
2. 로그인 후 **"Add new site"** → **"Deploy manually"** 클릭
3. 이 폴더(`From Web to App_py`) 전체를 드래그해서 올립니다
4. 1분 안에 배포 완료! 주소가 자동으로 생깁니다
5. 이 주소를 강의 때 브라우저에서 열면 됩니다

### GitHub Pages 사용 (무료)
1. GitHub에 저장소를 만듭니다
2. 파일을 전부 올립니다
3. Settings → Pages → Source를 `main` branch로 선택
4. 주소: `https://[아이디].github.io/[저장소이름]`

## 11. 문제 해결

| 문제 | 해결 방법 |
|---|---|
| 도구를 눌러도 반응이 없어요 | 브라우저 팝업 차단 해제 (주소창 옆 아이콘 클릭 → 허용) |
| 새 창이 너무 작아요 | 폰 종류를 iPad Mini로 바꿔보세요 |
| 즐겨찾기가 사라졌어요 | 브라우저 데이터를 지우면 초기화됩니다. 다시 추가해주세요 |
| 프로젝터에 안 나와요 | 노트북 화면 설정에서 "확장" 또는 "복제" 모드 확인 |
| 다른 PC에서도 쓰고 싶어요 | Netlify에 올리면 어디서든 같은 주소로 접속 가능합니다 |
| 핫스팟 연결이 느려요 | 수강생이 영상 스트리밍을 끄도록 안내 |
