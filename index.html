🍎 아이폰 화면을 PC에 띄우기

먼저 PC에 수신 프로그램을 설치하세요:

📥 LonelyScreen 설치 (무료) 📥 AirServer 설치 (유료)

설치 후 아이폰 연결 순서:

  1. PC에서 LonelyScreen (또는 AirServer) 실행
  2. 아이폰과 PC를 같은 와이파이에 연결
  3. 아이폰 오른쪽 위를 아래로 쓸어내려 제어센터 열기
  4. "화면 미러링" 버튼 누르기
  5. 목록에서 PC 이름 선택하면 끝!
  6. 프로젝터에 아이폰 화면이 그대로 나와요

🤖 안드로이드 화면을 PC에 띄우기

방법 1: USB 케이블 연결 (가장 안정적, 추천!)

📥 scrcpy 설치 (무료)
  1. 위 버튼 클릭 → scrcpy-win64-vX.X.zip 다운로드
  2. 압축 풀기 → scrcpy.exe 실행
  3. USB 케이블로 폰과 PC 연결
  4. 폰에서 "USB 디버깅 허용" 누르기
  5. PC에 폰 화면이 바로 나와요!

방법 2: 무선 연결 (삼성 등)

  1. 화면 위에서 아래로 내려서 빠른 설정 열기
  2. "Smart View" 또는 "화면 미러링" 누르기
  3. 목록에서 연결할 기기 선택하면 끝!

💻 이 프로그램 활용법

이 프로그램은 노트북/PC 브라우저에서 실행합니다.
별도 설치가 필요 없고, 파일을 열기만 하면 바로 사용할 수 있어요.

방법 1: 파일로 바로 열기
1. 이 폴더의 index.html 파일을 더블클릭
2. 브라우저에서 바로 열림 → 프로젝터에 연결해서 사용

방법 2: 인터넷에 올려서 쓰기 (추천)
1. Netlify(무료)에 이 폴더를 올리면 주소가 생김
2. 어떤 PC에서든 그 주소로 접속하면 바로 사용 가능
3. 올리는 방법: netlify.com 가입 → "Add new site" → 폴더 드래그

🖥️ 강의실에서 노트북 세팅 순서

  1. 노트북을 프로젝터/대형 모니터에 연결
  2. 브라우저(Chrome 추천)에서 이 프로그램 열기
  3. 왼쪽 패널에서 "📲 앱 설치" 탭 선택
  4. 수강생에게 프로젝터 화면을 보면서 앱 설치하도록 안내
  5. 설치 완료 후 "📱 폰화면" 탭으로 전환
  6. "자주 쓰는 도구"에서 도구 선택 → 폰 크기 새 창이 열림
  7. 열린 새 창을 프로젝터로 보여주며 실습 진행

📱 강의 중 폰으로 직접 시연할 때

  1. 왼쪽 패널에서 "🔗 폰 띄우기" 탭 선택
  2. 안내에 따라 폰 화면을 노트북에 미러링
  3. 프로젝터에 폰 화면이 그대로 나옴
  4. 강사가 폰에서 직접 조작 → 수강생이 보고 따라함

📶 와이파이가 안 될 때 - 핫스팟 연결

강의실 보안 와이파이가 연결 안 되거나, 인터넷이 안 될 때
강사 폰의 핫스팟으로 수강생 전원이 연결할 수 있어요.

🍎 아이폰 핫스팟 켜기
  1. 설정 → 개인용 핫스팟 열기
  2. "다른 사람의 연결 허용" 켜기
  3. 와이파이 이름과 암호가 화면에 표시됨
  4. 수강생에게 이름/암호를 알려주고 와이파이 연결하도록 안내
🤖 안드로이드 핫스팟 켜기
  1. 화면 위에서 아래로 내려서 빠른 설정 열기
  2. "모바일 핫스팟" 또는 "테더링" 버튼 누르기
  3. 또는: 설정 → 연결 → 모바일 핫스팟 및 테더링
  4. 핫스팟 이름과 비밀번호를 수강생에게 안내
💡 핫스팟 사용 팁
  • 데이터 요금제가 충분한지 먼저 확인하세요
  • 수강생이 10명 이상이면 데이터 소모가 클 수 있어요
  • 영상 재생은 피하고, 텍스트 위주 실습을 추천해요
  • 무제한 요금제라면 걱정 없이 사용 가능!
  • 보조 배터리를 준비하면 더 안심이에요

📶 핫스팟으로 노트북도 같이 쓰기

강사 폰 핫스팟에 노트북도 연결하면,
노트북에서 이 프로그램을 쓰면서 인터넷도 사용할 수 있어요.

  1. 강사 폰에서 핫스팟 켜기 (위 방법 참고)
  2. 노트북 와이파이에서 핫스팟 이름 찾아서 연결
  3. 노트북 브라우저에서 이 프로그램 열기
  4. 수강생도 같은 핫스팟에 연결
  5. 모두 인터넷 사용 가능! → 실습 진행
✅ 요약: 핫스팟 하나로 강사 노트북 + 수강생 폰 전부 인터넷 연결 가능
